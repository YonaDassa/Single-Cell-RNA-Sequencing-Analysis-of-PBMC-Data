# PBMC Single-Cell RNA-seq Analysis

This repository contains a workflow for analyzing single-cell RNA sequencing (scRNA-seq) data from human PBMCs.

<img width="677" height="426" alt="image" src="https://github.com/user-attachments/assets/66e4211a-5f03-4504-9514-26442d61aa0c" />

## Analysis Steps

1. Data loading and quality control
2. Normalization and feature selection
3. Dimensionality reduction (PCA, UMAP, t-SNE)
4. Clustering
5. Marker gene identification
6. Cell type annotation using SingleR

## Requirements

- R >= 4.2
- Seurat, SingleR
